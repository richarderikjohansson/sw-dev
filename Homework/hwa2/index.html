The two functions were created in two different modules and could successfully be imported both within the package, and also outside this package. The reverse list function manipulates the list in place with:

list_to_reverse[::-1]
The second function import this function and reverse the first list. I then kept it simple by looping through the two lists with zip() and appending a new list with the lists method. Very easy and maybe not the fastest and most reliable thing to do this since the lists must be of equal length.

Created two functions, one to write a string to a binary file, and one to read a binary file. The string is encoded with utf-8 encoding by default.

def interleave_two_lists(list1: list, list2: list) -> list:
    """Interleave two lists

    Args:
        list1: List to reverse
        list2: Other list

    Returns:
        List with interleaved from the two lists
    """

    reversed_list = reverse_list(list1)
    new_list = [str]

    for pair in zip(reversed_list, list2):
        new_list.extend(pair)

    return new_list

def reverse_list(list_to_reverse: list) -> list:
    """Reverse a list

    Args:
        list_to_reverse: List to reverse

    Returns:
        Reversed list
    """
    reversed_list = list_to_reverse[::-1]
    return reversed_list
